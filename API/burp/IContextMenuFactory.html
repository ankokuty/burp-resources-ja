インタフェース IContextMenuFactory


  • public interface IContextMenuFactory
    拡張はこのインタフェースを実装して、IBurpExtenderCallbacks.registerContextMenuFactory()を呼び出すと、カスタムコンテキストメニューアイテムのファクトリーを登録できます。

    • メソッドの詳細

      • createMenuItems

        java.util.List<javax.swing.JMenuItem> createMenuItems(IContextMenuInvocation invocation)
        ユーザがBurp内の任意のコンテキストメニューを呼び出したときに呼ばれます。ファクトリーは、メニュー呼び出しの詳細に基づいて、コンテキストメニューに表示されるカスタムコンテキストメニューアイテムを提供します。
        パラメータ:
        invocation - IContextMenuInvocationインタフェースを実装し、拡張がコンテキストメニュー呼び出しの詳細を照会できるオブジェクト。
        戻り値:
        表示する必要があるカスタムメニューアイテム(サブメニュー、チェックボックスメニューアイテムなどを含む)のリスト。メニューアイテムが必要ない場合は、拡張はこのメソッドでnullを返します。
