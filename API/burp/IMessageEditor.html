パッケージ burp

インタフェース IMessageEditor

  • public interface IMessageEditor
    独自のUIを使う拡張の、BurpのHTTPメッセージエディタのインスタンスを提供します。拡張がこのインタフェースのインスタンスを取得するには、IBurpExtenderCallbacks.createMessageEditor()を呼び出してください。

        現在のメッセージがユーザによって変更されているかどうかを判断します。

      • getSelectedData

        byte[] getSelectedData()
        ユーザによって現在選択されているデータを返します。
        戻り値:
        ユーザによって現在選択されているデータ、または選択されていない場合はnull。

      • getSelectionBounds

        int[] getSelectionBounds()
        該当する場合、表示されているメッセージにおけるユーザの選択範囲を取得します。
        戻り値:
        表示されたメッセージでユーザが選択した、開始と終了オフセットを含んだint[2]配列。ユーザが現在のメッセージで選択をしていない場合は、両方のオフセットはエディタ内のキャレットの位置を示します。エディタビューでメッセージ内で選択の概念が適用されない場合は、nullを返します。
