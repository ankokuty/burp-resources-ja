burp

インタフェース IIntruderPayloadGenerator


  • public interface IIntruderPayloadGenerator
    カスタムIntruderペイロードジェネレータに使用されます。IIntruderPayloadGeneratorFactoryを登録した拡張は、新規Intruder攻撃の一部として必要になった場合に、このインタフェースの新規インスタンスを返す必要があります。

    • メソッドの詳細

      • hasMorePayloads

        boolean hasMorePayloads()
        ペイロードジェネレータがさらにペイロードを提供できるかBurpが判断する際に使用します。
        戻り値:
        全ての利用可能なペイロードを使い切った場合はfalse返し、それ以外はtrueを返してください。

      • getNextPayload

        byte[] getNextPayload(byte[] baseValue)
        Burpに次のペイロード値を取得させます。
        パラメータ:
        baseValue - 現在のペイロードポジションの元値。元値の概念が利用できない場合(例えば破城槌(Battering ram)攻撃など)、nullを返します。
        戻り値:
        攻撃で次に使用するペイロード。

      • reset

        void reset()
        ペイロードジェネレータの状態をリセットし、次にgetNextPayload()を呼んだ際に最初のペイロードを再び返させるために、Burpが使用します。複数のペイロード位置に対して同じペイロードジェネレータを使用するスナイパー攻撃などの場合、このメソッドが呼び出されます。
