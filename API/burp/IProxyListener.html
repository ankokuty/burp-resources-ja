パッケージ burp

インタフェース IProxyListener

  • public interface IProxyListener
    拡張はこのインタフェースを実装して、IBurpExtenderCallbacks.registerProxyListener()を呼び出すと、Proxyリスナーに登録できます。Proxyツールが処理しているリクエストとレスポンスがリスナーに通知されます。Proxyリスナーを登録すると、これらのメッセージの解析や変更を拡張が実行できます。

      • processProxyMessage

        void processProxyMessage​(boolean messageIsRequest,
                         IInterceptedProxyMessage message)
        HTTPメッセージがProxyに処理されているときに呼び出されます。
        パラメータ:
        messageIsRequest - メッセージがリクエストかレスポンスかを示すフラグ。
        message - 拡張からメッセージ詳細の照会や更新を行い、メッセージをインターセプトしたり手動レビューや編集用にユーザに表示するかどうかを制御する際に使用する、IInterceptedProxyMessageオブジェクト。
