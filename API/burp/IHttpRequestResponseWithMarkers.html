インタフェース IHttpRequestResponseWithMarkers

  • すべてのスーパーインタフェース:
    IHttpRequestResponse

    public interface IHttpRequestResponseWithMarkers
extends IHttpRequestResponse
    マーカが適用されたIHttpRequestResponseオブジェクトに使用します。拡張は、このインタフェースのインスタンスをIBurpExtenderCallbacks.applyMarkers()を使用して作成するか、独自の実装を提供できます。Intruderのペイロードポジションの指定や、Scannerの挿入ポイント、Scanner問題の強調表示など、様々な場面でマーカが使用されます。

      • getRequestMarkers

        java.util.List<int[]> getRequestMarkers()
        リクエストマーカの詳細を返します。
        戻り値:
        リクエストメッセージのマーカのオフセットを示す、インデックスペアのリスト。リスト内の各アイテムは、マーカの開始オフセットと終了オフセットを含むint[2]配列です。リクエストマーカが定義されていない場合、nullを返します。
