Burp Suiteモバイルアシスタントをインストール

注: Burp Suiteモバイルアシスタントは、機能の性質上、脱獄されたデバイスにインストールする必要があります。脱獄では通常、モバイルアシスタントのインストールに使用できる、人気パッケージマネージャーのCydiaをインストールします。上級ユーザーはモバイルアシスタントを直接インストールしたいかもしれません。例えば、http://[burp-host-ip]:8080/mobileassistant.deb のように、Burp Suiteのブラウザ内インタフェースにブラウザでアクセスすると可能です。Cydiaは厳密には必要ありませんが、モバイルアシスタントはCydia Substrateに依存しています。

CydiaでBurp Suiteモバイルアシスタントをインストールするには次の手順を実行してください:

  1. 脱獄したデバイスからネットワークアクセスできるシステムでBurp Suiteを起動し、プロキシーリスナーがアクセスできるネットワークインタフェースとポートをバインドしているか確認します。
  2. 脱獄したデバイスで、Cydiaを開きSourcesタブに移動します。
  3. 右上隅にあるEditボタンをタップします。
  4. 左上隅にあるAddボタンをタップします。
  5. BurpプロキシーリスナーのIPアドレス(またはホスト名)とポート番号と、プロトコルにはHTTPを使用し、http://[burp-host-ip]:8080 のようにAPT URLプロンプトに入力します。Cydiaが接続できない場合は、関連するネットワークインタフェースをリッスンするようBurpプロキシーリスナーが設定されているか、またコンピューターのホストファイアウォールや無線ネットワークのデバイス分離で接続がブロックされていないか確認してください。
  6. 終了するには、Doneボタンをタップします。
  7. Cydiaのパッケージソースから通常の方法でモバイルアシスタントをインストールします。モバイルアシスタントを検索するか、新しいソースのパッケージをブラウズしてInstallをタップします。
