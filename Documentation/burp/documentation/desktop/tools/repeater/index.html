Burp Repeaterとは?

Burp Repeaterは、興味深いHTTPメッセージやWebSocketメッセージを何度も変更して送信できるツールです。

Repeaterは、次のようなあらゆる用途に使用できます:

  • 入力ベースの脆弱性をテストするために、パラメータ値を変化させたリクエストを送信する。
  • 特定の順序で一連のHTTPリクエストを送信し、多段階処理中の脆弱性や、接続状態の操作に依存する脆弱性をテストする。
  • Burp Scannerから報告された問題を手動で検証する。

Repeaterでは、複数のメッセージをそれぞれのタブで同時に作業できます。メッセージに行った変更は、タブの履歴に保存されます。グループ機能により、大量に開いたタブを簡単に管理できます。
