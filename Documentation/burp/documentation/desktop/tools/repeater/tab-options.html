タブ固有のオプション設定

Repeaterメニューで選択したオプションを、個々のタブで上書きできます。

タブ固有のオプションを設定するには:

  1. 設定したいタブを選択します。
  2. 送信ボタンの横にある設定アイコンをクリックすると、Repeaterオプションのコンテキストメニューが表示されます。これらのオプションは、グローバルのRepeaterメニューと同じです。
  3. メニューから必要なオプションを選択します。

タブ固有のメニューでオプションを選択した場合、Repeaterはそのタブですべてのグローバルオプションを無視します。タブからリクエストを送信する前に、タブ固有のメニューですべてのオプションが正しく設定されているかを確認する必要があります。たとえば、グローバルRepeaterメニューでリダイレクト中にcookieを処理するを選択し、タブ固有のメニューでHTTP/1接続を再利用するを選択すると、グローバルメニューのリダイレクト中にcookieを処理するは無視されます。

タブのオプションを変更すると、そのタブの設定アイコンが青くなります。タブをグローバルRepeater設定に戻すには、タブの設定アイコンをクリックし、コンテキストメニューからグローバルデフォルトを復元を選択します。

プロジェクトファイルを使用している場合、開いているタブに設定したタブ固有のオプションは、Burp Suiteを再度開いたときにも保持されます。
