タブグループの管理

タブをグループ分けすると、大量に開いたタブを管理しやすくなり、複数のタブから順番にリクエストも送信できます。

注

グループ化されたリクエストを順番に送信する方法は、リクエストを順番に送信を参照してください。

新しいタブグループの作成

タブグループを作成するには:

  1. 新規グループ作成をクリックすると、グループ作成のコンテキストメニューが表示されます。新規グループ作成オプションには、複数の場所からアクセスできます:

    • タブ追加(+)メニュー
    • タブバーのオプションメニュー
    • タブを右クリックしたときに表示される、グループにタブを追加コンテキストメニューオプション
  2. グループ名を入力します。
  3. チェックボックスを使って、グループに追加するタブを選択します。必要に応じて、Shiftキーを押しながらクリックすると、複数のタブを選択できます。
  4. グループカラーを選択します。グループのタブは、タブバー上でこの色で強調表示されます。
  5. 作成をクリックするとコンテキストメニューが閉じ、グループが作成されます。

タブバーにグループのアイコンが追加されます。アイコンをクリックすると、グループ内の個々のタブが表示されます。

作成したタブグループは、Burp Suiteを再起動しても開いたままです。

既存のグループを編集する

既存のグループは、次の方法で編集できます:

  • 既存のグループの詳細を編集するには、そのアイコンを右クリックし、グループの編集を選択し、グループの編集コンテキストメニューを表示します。このメニューの入力欄は、グループ作成時に使用したものと同じです。
  • グループにタブを追加するには、タブを右クリックし、グループにタブを追加メニューからグループを選択します。
  • グループからタブを削除するには、タブを右クリックし、コンテキストメニューからグループからタブを削除を選択します。グループから最後のタブを削除すると、そのグループは自動的に閉じられます。
  • タブを開いたままグループを閉じるには、グループのアイコンを右クリックし、タブのグループ解除を選択します。

タブグループを閉じる

グループとそのすべてのタブを閉じるには、グループのアイコンを右クリックし、コンテキストメニューからグループ全体の削除を選択します。
