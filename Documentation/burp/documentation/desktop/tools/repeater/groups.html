タブグループの管理

タブをグループ分けして、大量に開いたタブを管理できます。また、複数のタブから順番にリクエストを送信できます。

注

グループ化されたリクエストを順番に送信する方法は、リクエストを順番に送信を参照してください。

新しいタブグループの作成

タブグループを作成するには:

  1. タブの追加ボタンをクリックし、タブグループ作成を選択すると、グループ作成のコンテキストメニューが表示されます。
  2. グループ名を入力します。
  3. チェックボックスを使って、グループに追加するタブを選択します。Shiftキーを押しながらクリックすると、複数のタブを選択できます。
  4. グループカラーを選択します。グループのタブは、タブバー上でこの色で強調表示されます。
  5. 作成をクリックしてグループを設定します。タブバーにグループのアイコンが追加されます。アイコンをクリックすると、グループ内の個々のタブが表示されます。

作成したタブグループは、Burp Suiteを再起動しても開いたままです。

既存グループの編集

既存のタブグループを編集できます:

  • グループ編集 - グループを右クリックして表示されるコンテキストメニューで、グループ編集を選択します。このメニューのフィールドは、グループを作成するときに使用したフィールドと同じです。
  • グループにタブを追加 - タブを右クリックし、グループにタブを追加メニューでグループを選択します。
  • グループからタブを削除 - タブを右クリックし、グループからタブを削除を選択します。グループから最後のタブを削除すると、そのグループは自動的に閉じられます。
  • グループ内の選択したタブ以外を閉じる - タブを右クリックし、グループ内の他のタブを閉じるを選択します。
  • タブを開いたままグループを閉じる - グループのアイコンを右クリックし、タブのグループ解除を選択します。

タブグループを閉じる

グループとそのすべてのタブを閉じるには、グループのアイコンを右クリックし、グループ全体の削除を選択します。
