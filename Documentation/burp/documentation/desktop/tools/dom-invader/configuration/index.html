DOM Invaderの設定

DOMベースの脆弱性をテストすると、テスト対象のWebサイトが正しく動作しなくなる副作用が発生することがあります。DOM Invaderは高度な設定が可能で、最良の結果を得るために動作を微調整できます。

DOM Invaderの設定メニューにアクセスするには、ブラウザの右上にあるBurp Suiteのロゴをクリックし、DOM Invaderタブに切り替えます。
