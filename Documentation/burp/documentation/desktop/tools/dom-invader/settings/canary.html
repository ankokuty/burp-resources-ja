DOM Invaderのカナリア設定

DOM Invaderの設定メニューで、DOM Invaderが対象サイトに挿入するカナリア文字列の変更や更新ができます。

DOM Invaderの設定メニューにアクセスするには、ブラウザの右上にあるBurp Suiteのロゴをクリックし、DOM Invaderタブに切り替えます。

設定メニューの下部には、DOM Invaderがトラッキングしているカナリア文字列が表示されます。次の選択肢があります。

カナリアをコピー

コピーをクリックすると、現在のカナリア文字列をクリップボードにコピーできます。これは、それぞれのソースに手動で文字列を挿入する場合に便利です。

カナリアの変更

ランダムに生成されたデフォルトのカナリアはいつでも変更でき、代わりに独自のカスタムカナリア文字列を追跡できます。

カナリアを変更するには:

  1. デフォルトのカナリアの代わりに使用する文字列を入力します。あるいは、Randomizeをクリックすると、新たなランダム文字列が生成されます。

  2. Update canaryをクリックします。

  3. Reloadをクリックすると、ブラウザが更新されます。これは、変更を有効にするために必要です。これでDOM Invaderは、代わりに新しいカナリア文字列を追跡するようになります。

注

誤検出を避けるためには、ページ上で自然に発生する文字列を使用しないでください。
