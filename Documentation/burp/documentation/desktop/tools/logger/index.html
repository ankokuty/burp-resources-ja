Burp Logger

Burp Loggerは、Burp Suiteが生成するすべてのHTTPトラフィックをリアルタイムで記録します。Loggerを使用すると次ができます:

  • Burpのツールや拡張から送信されたリクエストの調査。
  • Burp Scannerが送信したリクエストをリアルタイムで見る。
  • 拡張の動作を検証する。
  • セッションハンドリングルールによる変更を調査する。

デフォルトでLoggerは、Burp Suiteのすべてのツールによって生成されたトラフィックを表示します。これには、変更されたリクエスト、および拡張によって送信されたリクエストが含まれます。一方HTTP履歴は、Burpを経由してプロキシされたブラウザからのトラフィックのみを表示します。
