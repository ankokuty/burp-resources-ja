Logger

Loggerは、ネットワークアクティビティを記録するツールです。Loggerは、調査や分析のために、Burp Suiteが生成するすべてのHTTPトラフィックを記録しますが、特に次の場合に便利です:

  • Burp Suiteの結果が予期しない結果だったとき、何が起こったかを調査する場合。
  • 作業にセッション処理が含まれるとき、Burp Suiteが送信する内容の詳細を確認する場合。
  • 長期的に進行中のタスク(バックグラウンドスキャンなど)がまだ実行されているか確認する場合。
  • Burp Suiteが行っていることを可視化する必要がある問題を分析する場合。

ログ出力とメモリ

Loggerは、割り当てられたメモリ制限いっぱいまでHTTPネットワークトラフィックの記録を保持します。制限に達した後、新しいエントリが作成されると、Loggerは最も古いエントリを破棄します。Loggerのデフォルトのメモリ制限は50MB、またはBurp Suiteに1GB以上のメモリを割り当てた場合は100MBです。キャプチャフィルタをクリックすると、メモリ制限を変更できます。ログ出力に大量のメモリを割り当てると、パフォーマンスの問題が発生する可能性がある点に注意してください。デフォルトのメモリ量でログ出力を開始し、大量のエントリを必要とする問題がある場合にのみ増やすことを推奨します。

Logger機能

Loggerエントリは読み取り専用です(注目する項目に注釈を付けること以外:以下参照)が、それ以外はメッセージエディタのすべての機能が有効で、Inspectorも使えます。Loggerエントリを、RepeaterやIntruderなど他のツールに送信もできます。

エントリに関する情報が、複数の列に表示されます。すべての列が各エントリに関連するわけではありません。列ドロップダウンメニューをクリックすると、表示する列を調整できます。列ヘッダのクリックや列の境界線をドラッグすると、列のソートやサイズ変更ができます。

#でマークされた左端の列は、このセッションでログに記録されたエントリ数をカウントします。すべてのエントリをキャプチャしないよう選択した場合(オプション参照)、カウントが連続しない可能性があります。カウントは、実際にキャプチャされたイベントではなく、キャプチャされる可能性のあるイベント数を参照しているためです。

ログをクリアアイコンをクリックすると、現在のすべてのエントリを削除します。

後で確認できるようにエントリにマークが付けられます。エントリの#列をクリックすると、そのエントリの色を変えて強調表示できます。エントリのコメント列のダブルクリックすると、そのエントリにコメントを残せます。

LoggerエントリのCSVエクスポート

LoggerエントリをCSVファイルとしてエクスポートするには、関連するエントリをテーブルで選択し、右クリックしてCSVとしてエクスポートを選択します。0または1つのエントリを選択している場合は、テーブル全体がエクスポートされることに注意してください。

エントリをCSV形式でエクスポートする場合、Loggerは特定のデータを次のようにエンコードします:

  • 日時は、UTCオフセットでISO 8601フォーマットになります: yyyy-MM-dd'T'HH:mm:ss.SSS'Z'。
  • バイナリデータ(HTTP/2リクエストやレスポンスなど)は、Base64エンコードされた文字列になります。
  • カンマを含むデータは、データをダブルクオートで囲みエスケープされます。たとえば、a,b は "a,b" になります。
  • ダブルクオートを含むデータは、ダブルクオートを追加してエスケープされます。たとえば、"a" は """a""" になります。
  • -、+、=、@で始まるデータは、Excel用にシングルクオート(')を前に付けてエスケープされます。

タスクロガー

Loggerは、Burp Suite全体のトラフィックをキャプチャします。さらにタスクロガーを使用すると、単一のタスクによって生成されたトラフィックを調べられます。タスクログのエントリを調べるには、次の手順を実行します:

  • ダッシュボードから、タスクの詳細表示アイコンをクリックします。
  • タスクウィンドウ内のLoggerタブを選択します。

タスクロガーはBurp Loggerと同じ機能を備えていますが、タスクごとに個別にメモリを割り当てます。各タスクをログ出力するためのデフォルトのメモリ割り当ては10MB、またはBurp Suiteに1GB以上のメモリアクセスを許可した場合は20MBです。タスクロガーを使用すると、予期しない動作をしているタスクの調査や、タスクの進行状況を監視できます。

タスクロガーにもBurp Loggerと同様のキャプチャと表示オプションがありますが、ツールでキャプチャやツールでフィルタオプションはありません。これは、タスクに使用されるツールからのトラフィックのみをキャプチャして表示するためです。
