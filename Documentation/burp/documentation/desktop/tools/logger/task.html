タスクLogger

単一のタスクによって生成されたトラフィックを調べられます。これにより、予期せぬ動作をしたタスクの調査や、特定のタスクの進捗状況を監視できます。

タスク固有のログエントリを表示するには:

  1. ダッシュボードに移動し、タスク欄で該当するタスクまでスクロールします。
  2. タスクの詳細表示アイコンをクリックします。タスクの詳細が表示されるウィンドウが開きます。
  3. タスクウィンドウ内のLoggerタブを選択します。

タスク固有のLoggerはBurp Loggerと同様の機能を持ちますが、注目する違いがいくつかあります。

  • 各タスクのデフォルトのメモリ割り当ては10MB、またはBurp Suiteに1GB以上のメモリアクセスを許可した場合は20MBです。
  • タスク固有のLoggerはタスクに使用されたツールからのトラフィックのみをキャプチャして表示するため、ツールによるキャプチャまたはフィルタリングはできません。

タスク固有のLoggerがキャプチャや表示する内容を編集する方法や、メモリ割り当て方法などの詳細は、Burp Loggerオプションを参照してください。
