Burp Intruder攻撃結果

Burp Intruder攻撃を開始すると、新しい結果ウィンドウで攻撃が実行されます。ここには、攻撃結果と、現在の攻撃の元となった各設定タブの複製があります。

通常、次のような方法で興味深いレスポンスを確認できます。

  • 異なるHTTPステータスコード。
  • レスポンス長の差異。
  • 特定の表現の有無。
  • エラーまたはタイムアウトの発生。
  • レスポンスの受信または完了までにかかる時間。

Intruderは、結果を分析し興味深いアイテムを特定するためのさまざまな機能を備えています。その後、テストワークフローの一部として、さらなるアクションを実行できます。
