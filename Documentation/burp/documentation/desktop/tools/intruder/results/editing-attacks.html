攻撃の編集

攻撃実行中に、攻撃の制御や設定の変更ができます。これは、結果ウィンドウから行います。

攻撃の制御

結果ウィンドウを開いた状態で、トップレベルの攻撃メニューから攻撃を制御できます。これにより、次の操作ができます:

  • 攻撃の一時停止。
  • 攻撃の再開。
  • 攻撃の再実行。

関連ページ

  • 攻撃を保存して閉じる方法は、攻撃の保存を参照してください。

攻撃の設定

結果ウィンドウには、元の設定タブを複製したタブがあります。これらを使って、攻撃実行中にほとんどの攻撃設定の確認や変更ができます。

攻撃設定の基本的な部分は変更できません。これは、次の通りです:

  • 攻撃タイプ。
  • ペイロードポジション。
  • ペイロードタイプ。

これらの値を変更したい場合は、元の攻撃タブで編集してから、新しい攻撃を開始してください。

注

設定を変更する前に、攻撃の一時停止を推奨します。攻撃の実行中に設定を変更する際に各キーが押されるたびに変更が実行されるため、予期せぬ影響を与える可能性があります。

たとえば、数値ペイロードタイプを使っているときTo欄で数字を削除すると、To欄の数字が小さくなるため、攻撃が突然完了するかもしれません。
