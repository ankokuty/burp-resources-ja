ペイロード

1 つまたは複数のペイロードセットを設定します。ペイロードセットの数は、ポジションタブで定義されている攻撃の種類に依存します。パラメータのファジング、ユーザパスワードの推測、ページ識別子の巡回のような一般的なタスクでは、１つのペイロードセットのみが必要になります。

ペイロードセットの設定に必要な手順は次の通りです:

  • 設定したいペイロードセットをドロップダウンリストから選択します。
  • 使用するペイロードタイプをドロップダウンリストから選択します。多くのペイロードタイプが利用でき、高度に設定可能で、あらゆる状況のペイロードの生成を自動化できます。
  • 選択したペイロードタイプのペイロードオプションを設定します。
  • 様々な方法で操作するペイロード処理ルール、正しい文字をURLエンコードしてHTTP経由で安全に送信できるようにするエンコーディングなど、必要なペイロード処理を設定します。
