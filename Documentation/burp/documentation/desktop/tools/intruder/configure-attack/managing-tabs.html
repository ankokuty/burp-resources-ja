Burp Intruder攻撃タブの管理

Burp Intruderにリクエストを送信すると、新しい攻撃タブが作成され、リクエストの詳細が自動的に入力されます。

タブヘッダ上のコントロールを使用して、新しいタブの作成や、既存のタブに対するさまざな変更ができます:

  • 攻撃タブを作成する - タブを追加ボタン をクリックすると、新しいタブが開きます。新しいタブを開いた際の設定は、トップのIntruderメニューで、次から選択できます。

    • デフォルトのタブ設定を使用。
    • 最初のタブから設定をコピー。
    • 最後のタブから設定をコピー。
  • タブ名変更 - タブヘッダをダブルクリックし、新しいタブ名を入力します。

  • タブ表示の切り替え - タブを右クリックし、タブ表示オプションを選択します。2種類の表示方法から選択できます:

    • スクロール表示 - スクロール可能な1列に表示されます。すべての開いているタブをドロップダウンリストで表示するには、検索アイコンをクリックします。
    • 折り返し表示 - 複数の行に折り返して、すべてを画面上に表示します。

  • タブを閉じる - 複数の方法でタブを閉じられます:

    • 1つのタブを閉じる - タブヘッダのそばにある閉じるアイコンをクリックします。
    • すべてのタブを閉じる - オプションメニューを選択し、すべてのタブを閉じるを選択します。
    • 選択したタブ以外のタブを閉じる - タブを右クリックし、他のタブを閉じるを選択します。
    • 選択したタブの片側にあるすべてのタブを閉じる - タブを右クリックし、左側のタブを閉じるまたは右側のタブを閉じるを選択します。
  • 最後に閉じたタブを再び開く - 任意のタブを右クリックし、閉じたタブを開くを選択します。
