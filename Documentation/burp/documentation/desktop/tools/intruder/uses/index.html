Burp Intruderの代表的な使い方

Burp Intruderは、Webアプリケーションをテストする際のすべてのタイプのタスクを自動化できる、柔軟性の高いツールです。Intruderの最も一般的な使用例は、次のカテゴリに分類されます。

  • 識別子の列挙 - ユーザ名やパスワードなど、有効な識別子を抽出します。
  • 有用なデータの収集 - 識別子に関する興味深い情報を抽出します。
  • 脆弱性のファジング - クラッシュなどの例外を検出します。
