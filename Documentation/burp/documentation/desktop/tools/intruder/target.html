攻撃対象

攻撃対象サーバの詳細を設定します。必要なオプションは次の通りです:

  • ホスト - 対象サーバのIPアドレスまたはホスト名
  • ポート - HTTP/Sサービスのポート番号
  • HTTPSを使用する - TLSを使用するかどうか

これらの詳細を設定する最も簡単な方法は、攻撃したいリクエストをBurpの内のどこかで選び、コンテキストメニューの"Intruderに送信"オプションを選択します。すると選択したリクエストがIntruderの新しいタブに送信され、Targetとポジションタブが自動的に現れます。
