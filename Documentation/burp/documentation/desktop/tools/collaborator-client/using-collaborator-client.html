Burp Collaboratorクライアントの使用

Burp Collaboratorクライアントを使用すると、手動テスト時にBurp Collaboratorを利用できます。他のBurp Suiteツールのリクエストに含めて使用できるCollaboratorペイロードを生成し、それらのリクエストによって生じる通信についてCollaboratorサーバにポーリングできます。

クライアントを使用するには:

  1. Collaboratorタブを選択します。
  2. 必要に応じて、プラスボタンをクリックすると、新しいCollaboratorクライアントのサブタブが開きます。
  3. 生成したいペイロード数を入力します。
  4. 必要に応じてCollaboratorサーバの場所を含めるチェックボックスを使用し、Collaboratorサーバ完全なアドレスを含めるか、Collaborator IDだけを含めるかを定義します。
  5. クリップボードにコピーをクリックします。クライアントは、指定された数のペイロードをコピーします。
  6. ペイロードを挿入したいリクエストを開き、適切な場所に挿入します。
  7. リクエストを送信します。
  8. 再度、Collaboratorタブを開きます。
  9. 今すぐポーリングをクリックします。

クライアントは、生成されたペイロードに関連する通信が無いかCollaboratorサーバをポーリングし、通信の詳細をテーブルに表示します。

注

また、メッセージエディタでメッセージを右クリックし、コンテキストメニューからCollaboratorペイロードを挿入を選択すると、Collaborator IDを追加できます。システムは、最後に開いたCollaboratorクライアントタブから新しいIDを貼り付けます。
