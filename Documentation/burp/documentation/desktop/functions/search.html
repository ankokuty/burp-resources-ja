テキスト検索

Burpメニューの検索を選択すると、Suite全体からの検索が実行できます。対象サイトマップ内の選択したブランチ内からの検索も可能で、コンテキストメニューの関連ツールから検索を選択します。

検索ダイアログでは、次のオプションを設定できます:

  • 検索する文字列
  • 検索で大文字と小文字を区別するかどうか
  • 検索語がリテラル文字列か正規表現か
  • 一致しないもの(検索語を含まないアイテム)を検索し表示するかどうか
  • スコープ内のアイテムのみ検索するかどうか
  • Burpのツールで新たなHTTPメッセージが処理されたとき、検索結果を動的に更新するかどうか
  • HTTPメッセージ内で検索する箇所(リクエストとレスポンス、ヘッダとボディ)
  • 検索するツール

送信をクリックすると検索が開始し、検索結果の詳細がソート可能な一覧に表示されます。プレビューペインには、選択したアイテムの完全なリクエストとレスポンスが表示され、検索語にマッチした箇所が強調表示されます。コンテキストメニューから、他のツールへのリクエスト送信や、その他のアクションを実行できます。

コメントやスクリプトの検索

これらの機能で、対象サイトマップの一部またはすべてから、コメントやスクリプトを検索できます。サイトマップツリーの一部またはすべてを選択し、サイトマップのコンテキストメニューの関連ツールでコメント検索はたはスクリプト検索を選択すると、検索を開始できます。

検索ダイアログで、検索ボタンを使用して検索を実行します(または後で再実行します)。探索されたアイテムの詳細は、ソート可能な一覧に表示されます。プレビューペインには、選択したアイテムの完全なリクエストとレスポンスが表示され、関連する項目が自動的に強調捧持され、独自のタブにも抽出されます。コンテキストメニューから、他のツールへのリクエスト送信や、その他のアクションを実行できます。

動的に更新オプションを選択すると、Burpのツールで新たなHTTPメッセージが処理されたとき、結果を動的に更新します。エクスポートボタンで、すべてのスクリプトまたはコメントをファイルまたはクリップボードに保存でき、設定によって重複したアイテムを統合できます。

参照元検索

この機能で、特定のアイテムにリンクするHTTPレスポンスを、すべてのBurpツールから検索できます。この機能にアクセスするには、Burpの任意の場所か、TargetのサイトマップでHTTPリクエストを選択し、コンテキストメニューの関連ツールから参照元検索を選択します。

検索結果ウィンドウには、選択したアイテムにリンクしている(すべてのBurpツールの)レスポンスが表示されます。個々の検索結果を表示すると、どこからリンク参照されているか自動的にレスポンスがハイライトされます。

この機能は、リンクの検索時に元のURLをプレフィックスとして扱うため、ホストを選択するとそのホストへのすべての参照を、フォルダを選択するとそのフォルダ以下のアイテムへのすべての参照を検索できます。
