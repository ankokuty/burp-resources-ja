タスクスケジューラ

タスクスケジューラを使用すると、指定した時間及び間隔で、特定のタスクを開始及び停止できます。タスクスケジューラを使用すると、あなたが作業していない間に特定のタスクを自動で開始および停止したり、定期的または指定した時間に作業の保存ができます。

<a1>その他プロジェクトオプション</a1>のスケジュールタスクパネルから、新規タスクを追加できます。

新規タスクを作成すると、タスクの詳細を設定するウィザードが開きます。次のタスクが使用できます:

  • 自動タスク実行を一時停止する
  • 自動タスク実行を再開する

各タスクには開始時間を設定する必要があります。必要に応じて、定義された間隔でタスクを繰り返すように設定できます。
