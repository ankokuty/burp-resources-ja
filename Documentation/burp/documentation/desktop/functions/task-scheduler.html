タスクスケジューラ

タスクスケジューラを使用すると、指定した時間や間隔で特定タスクの開始や停止ができます。タスクスケジューラを使用すると、あなたが作業していない間に特定タスクを自動で開始や停止したり、定期的または指定した時間に作業を保存できます。

その他プロジェクトオプションのスケジュールタスクパネルから、新規タスクを追加できます。

新規タスクを作成すると、タスクの詳細を設定するウィザードが開きます。次のタスクが使用できます:

  • 自動タスク実行を一時停止する
  • 自動タスク実行を再開する

各タスクには開始時間を設定する必要があります。必要に応じて、定義された間隔でタスクを繰り返すように設定できます。
