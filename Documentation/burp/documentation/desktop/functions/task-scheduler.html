タスクスケジューラ

タスクスケジューラを使用すると、自動化されたタスクの実行を、指定した時間に一時停止や再開ができます。

タスクをスケジュールするには:

  1. 設定 > プロジェクト > タスクを選択すると、タスク設定メニューが開きます。
  2. 追加をクリックすると、スケジュールタスクダイアログが開きます。
  3. タスクをどのようにしたいかによって、タスク実行エンジンを一時停止するかタスク実行エンジンを再開するを選択します。次へをクリックします。
  4. タスクの実行時刻を、開始時間ドロップダウンメニューで選択します。
  5. タスクの実行日を、日付ドロップダウンメニューで指定します。
  6. このタスクが繰り返される頻度を、指定期間で繰り返し欄に指定します。この欄を空白のままにしておくと、タスクは一度のみ実行されます。
  7. 完了をクリックすると、タスクが保存されます。

タスク設定メニューで、既存のタスクに対して編集や削除もできます。
