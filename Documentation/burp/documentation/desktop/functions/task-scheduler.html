タスクスケジューラ

タスクスケジューラを使用すると、指定した時間及び間隔で、特定のタスクを開始及び停止できます。タスクスケジューラを使用すると、あなたが作業していない間に特定のタスクを自動で開始および停止したり、定期的または指定した時間に作業の保存ができます。

この機能にアクセスするには、Burp内の任意の場所かターゲットサイトマップの任意の部分でHTTPリクエストを選択し、"関連ツール"コンテキストメニューで"スケジュールタスク"を選択します。または、その他プロジェクトオプションのスケジュールタスクパネルから、直接新規タスクを追加できます。

新規タスクを作成すると、タスクの詳細を設定するウィザードが開きます。次のタスクが使用できます:

  • URLからスキャン
  • アクティブスキャンを停止
  • アクティブスキャンを再開
  • URLからSpider実行
  • 自動巡回停止
  • 自動巡回再開
  • ステート保存

タスクに応じて、URL(スキャンなど用)またはファイル(ステートの保存など用)の設定もできます。

各タスクには開始時間を設定する必要があります。必要に応じて、定義された間隔でタスクを繰り返すように設定できます。
