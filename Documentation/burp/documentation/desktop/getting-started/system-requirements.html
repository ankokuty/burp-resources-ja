Burp Suiteのシステム要件

Burp Suiteのシステム要件は、ソフトウェアの使用目的によって大きく異なります。一般的には、比較的低スペックのマシンでもほとんどの作業を行えますが、一部のユースケース(例えば、複数のスキャンを同時に実行する場合など)では、パフォーマンスに目立った影響を与えずに実行するために、かなりのパワーが必要な場合があります。

CPUコア/メモリ

  • 最小: 2コア, 4GB RAM - このスペックは、Webトラフィックのプロキシ処理やIntruderの攻撃など基本的なタスクに適しています。Burp Suiteはこれより低いスペックのマシンでも動作しますが、パフォーマンス上の理由から推奨できません。
  • 推奨: 2コア, 16GB RAM - これは、一般的な用途に適したスペックです。
  • 高度: 4コア, 32GB RAM - このスペックは、複雑なIntruder攻撃や大規模な自動スキャンなど、より激しいタスクに適しています。

空きディスク容量

  • 基本インストール: 1GB
  • プロジェクトファイルごと: 2GB

注:

プロジェクト用に2GBの空きディスク容量を推奨していますが、含まれるProxy履歴の量、実行したスキャン数、開いているRepeaterタブの数などの要因によって、これよりも大幅に大きくなる可能性があることに注意してください(数十GBになる可能性もあります)。

オペレーティングシステムとアーキテクチャ

Burp Suiteは、次のオペレーティングシステムの最新バージョンに対応しています:

  • Windows (Intel 64ビット)
  • Linux (Intel 64ビット)
  • OS X (Intel 64ビットおよびApple M1)
