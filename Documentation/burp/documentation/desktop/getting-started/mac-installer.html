どのBurp Suite Macインストーラをダウンロードする必要がありますか？

2020年、AppleはMacデバイス用の新タイプのプロセッサを導入し始めました。マシンに搭載されているプロセッサに応じて、異なるバージョンのBurp Suiteインストーラを使用する必要があります。

使用しているプロセッサを確認するには、画面の左上隅にあるAppleロゴをクリックし、このMacについてをたどってください。

このMacについてダイアログで、チップまたはプロセッサとマークされた行を探します。ここで、AppleかIntelかどちらのプロセッサを使用しているかがわかります。適切なBurp Suiteインストーラは、リリースページからダウンロードできます。
