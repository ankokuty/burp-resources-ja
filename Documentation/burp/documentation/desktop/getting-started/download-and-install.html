ダウンロードとインストール

ステップ1: ダウンロードする

Burp Suite ProfessionalやCommunity Editionの最新版は、次のリンクからダウンロードしてください。

ソフトウェアを選択してください

ProfessionalCommunity Edition

ステップ2: インストールする

インストーラを実行し、Burp Suiteを起動します。

プロジェクトファイルと設定を選択するように求められたら、次へとBurpを開始をクリックし、ひとまずこれをスキップします。

注

Burp Suite Professionalを使用している場合は、プロンプトが表示されたらライセンスキーを入力してください。まだ持っていない場合は、申し込みまたは無料トライアルのリクエストができます。

ステップ3: Burp Suiteの探索を開始する

Burp Suiteをまったく初めて使用する場合は、このチュートリアルの残りの部分に従って、コア機能についてのインタラクティブなガイド付きツアーを参照してください。
