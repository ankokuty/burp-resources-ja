Internet ExplorerへのBurpのCA証明書のインストール

BurpのCA証明書をインストールする前に、プロキシリスナーが動作しており、Burpが使われるようブラウザの設定ができているか確認します。

注: 以前、Burpが生成した別のCA証明書をインストールしている場合は、新しいものをインストールする前にそれを削除する必要があります。

Internet ExplorerにBurpのCA証明書をインストールするには、次の手順に従います:

Internet Explorerアイコンを右クリックし、"管理者として実行"を選択します。

Burpを実行している状態で、Internet Explorerでhttp://burpsuiteにアクセスします。CA証明書をダウンロードするためのリンクがある比較的わかりやすいWebページが表示されるはずです。表示されない場合は、プロキシのトラブルシューティングページを参照してください。何が悪かったかによっては、自動的にそこに遷移するかもしれません。

"CA Certificate"をクリックして、独自のBurp CA証明書をダウンロードします。保存する場所をメモしておいてください。

ダウンロードしたファイルを開きます。"証明書"ダイアログが開きます。"証明書のインストール"をクリックします。"証明書のインポート ウィザード"ダイアログで、"次へ"をクリックします。"証明書をすべて次のストアに配置する"を選択し、"参照"をクリックします。

"証明書ストアの選択"ダイアログで、"信頼されたルート証明機関"を選択し、"OK"をクリックします。

"次へ"、"完了"の順にクリックします。すべてのダイアログを閉じて、Internet Explorerを再起動します。Burpを実行したまま、HTTPSのURLにアクセスしてみてください。すべて正常に動作していれば、セキュリティ警告なしでページを参照できるはずです。

Internet ExplorerからBurpのCA証明書を削除

BurpのCA証明書をInternet Explorerから削除するには:

  1. Internet Explorerアイコンを右クリックし、"管理者として実行"を選択します。
  2. "インターネットオプション"を開き、"コンテンツ"タブで"証明書"をクリックします。
  3. "信頼されたルート証明機関"タブで、"PortSwigger CA"を選択し、"削除"をクリックします。
  4. プロンプトが表示されるたびに"はい"を選択し、Internet Explorerを再起動します。
