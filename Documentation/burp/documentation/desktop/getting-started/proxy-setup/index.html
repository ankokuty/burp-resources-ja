外部ブラウザで動作するようBurp Proxyを設定する方法

注: Burpを使用した手動テストに外部ブラウザを使用する場合にのみこれらの手順が必要です。必要に応じてBurpの内蔵ブラウザが使用でき、これは既にBurp Proxyで動作するように事前設定されています。内蔵ブラウザにアクセスするには、"プロキシ" > "インターセプト"タブに移動し、"ブラウザを開く"をクリックします。

BurpはWebプロキシサーバとして動作し、ブラウザと対象Webアプリケーションの中間者として機能します。Burpの設定済みChromiumブラウザの代わりに外部ブラウザを使用したい場合は、次の設定手順を行う必要があります:

  • 1 - プロキシリスナーがアクティブか確認する
  • 2 - Burp Suiteが動作するようブラウザを設定する
  • 3 - ブラウザのプロキシ設定を確認する
  • 4 - BurpのCA証明書をインストールする
