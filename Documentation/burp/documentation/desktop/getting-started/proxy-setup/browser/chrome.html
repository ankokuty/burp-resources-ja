Burpと連携するようChromeを設定する

他のブラウザとは異なり、Chromeには独自のプロキシ設定がありません。代わりに、コンピューターの内蔵ブラウザ(Windowsの場合はInternet Explorer、MacOSの場合はSafari)を設定すると、Chromeは自動的にこれらの設定を使用します。

ChromeでBurpを使用する場合は、オペレーティングシステムごとのプロキシ設定について次の手順を参照してください:

  • MacOS (Safari)
  • Windows (Internet Explorer)

内蔵プロキシの設定がどこにあるか分からない場合は、Chrome内からアクセスできます。

Chromeを開き、"カスタマイズ"メニューに移動します。"カスタマイズ"メニューで"設定"を選択し、"詳細設定"を開きます。

"詳細設定"セクションで、"パソコンのプロキシ設定を開く"または"プロキシ設定を変更する"ボタンをクリックします。すると、コンピュータのプロキシ設定が開くでしょう。
