Burp Suiteのダウンロードとインストール

Burp Suiteをダウンロードする手順、Burp SuiteプロフェッショナルとBurp Suiteコミュニティ版のどちらをインストールするかによって若干異なります。

Burp Suiteプロフェッショナル版のダウンロード

Burp Suiteの任意のバージョンのインストーラを、リリースページからダウンロードできます。ログインは不要です。これは、すでにライセンスキーを持っている場合に便利です。

ライセンスキーを持っていない場合は、最初に無料トライアルへの登録またはライセンスの購入を推奨します。アカウントを取得したら、ライセンスキーと、必要なプラットフォーム(Windows、MacOS、Linux)用の最新バージョンのBurp Suiteの両方を、あなたのアカウントページからダウンロードできます。

いずれかのネイティブプラットフォームのインストーラを使用する代わりに、JARファイルとしてダウンロードもでき、コマンドラインから直接起動できます。

Burp Suiteコミュニティ版のダウンロード

Burp Suiteコミュニティ版のダウンロードページで、無料のBurp Suiteコミュニティ版がダウンロードできます。その後、BurpSuiteのインストールに進めます。

Burp Suiteのインストール

Burp Suiteをインストールするには、インストーラを実行し、インストールウィザードでお好みのオプションを選択するだけです。このプロセスは、比較的自明なものでしょう。

Burp Suiteのシステム要件

Burp Suiteプロフェッショナルを最大限に活用するには、8GB以上のメモリと2つのCPUコアを搭載したマシンの使用を推奨します。大量の作業を実行している場合、または大規模や複雑なアプリケーションをテストしている場合には、これよりも多くのメモリが必要になる場合があります。使用しているコンピューターが適切かどうかわからない場合は、まずBurp Suiteコミュニティ版をテストしてみて、正しく機能するかどうかの確認を推奨します。

MacOSでインストーラ実行中のエラー

MacOSでインストーラを実行しようとすると、"Appleでソフトウェアが悪質なソフトウェアであることを確認できないため、インストーラを開けません"というエラーメッセージが表示される場合があります。この警告を回避するには、インストーラを信頼して実行するかどうか、手動で確認する必要があります。

  1. エラーメッセージのポップアップを閉じます。
  2. MacOSで、"システム環境設定" > "セキュリティとプライバシー"に移動します。
  3. ダイアログの下部に、Burp Suiteインストーラがブロックされたことを示すメッセージが表示されます。その横にある"とにかく開く"をクリックします。
  4. 最初のエラーメッセージが再びポップアップ表示されますが、今回は警告にもかかわらずインストーラを実行する"開く"ボタンがクリックできるようになります。
