Burp Suite(プロフェッショナルとコミュニティ版)入門

Burp Suiteを起動して実行するには、次の初期インストールと設定の手順を完了する必要があります。

  • 1 - Burp Suiteをダウンロードしてインストールする
  • 2 - Burp Suiteを起動し、起動オプションを選択する
  • 3 - Burpの事前設定されたブラウザを使用してテストを開始する
  • 4 - (オプション) 外部ブラウザで動作するようBurp Proxyを設定する

注: Burp Suiteを使うと、一部のアプリケーションで予期しない影響が出る可能性があります。機能や設定にしっかり慣れるまで、本番以外のシステムだけで使用してください。

次の手順

以上でBurp Suiteの初期設定は完了です。次に何をするかはあなた次第ですが、Burpとその主要なツールに慣れるために、以下のドキュメントの確認をお勧めします。

  • ウェブサイトをスキャン
  • ペネトレーションテストにBurp Suiteを使用する
  • Burp Suiteのツールでの作業
  • Burp Suiteの設定
  • 一般的な問題のトラブルシューティング
  • BurpのCA証明書のダウンロードに問題がありますか?
