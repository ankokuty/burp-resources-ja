設定の選択

スタートアップウィザードの次のステップは、プロジェクトに使用する設定ファイルの選択です。設定ファイルは、Burpのさまざまな側面のカスタム設定を保存するために使用されます。これは、特定のタイプのスキャン用のカスタム設定である場合もあれば、固有のユーザ設定やプロジェクト設定を含む設定ファイルである場合もあります。たとえば、プロジェクトの対象となるクライアントに応じて、プロジェクト設定が異なる場合があります。

Burpを初めて使用する場合は、最初にデフォルト設定を使用する必要があります。しかしBurpを継続して使用する場合は、独自の設定ファイルを保存して、他のプロジェクトで再利用できます。

プロジェクト設定を、次のオプションから選択できます:

  • Burpのデフォルトを使用する - Burpのデフォルトオプションを使用してプロジェクトを開きます。
  • プロジェクトに保存されたオプションを使用する - 既存のプロジェクトを再開する場合のみ使用でき、プロジェクトファイルに保存されたオプションを使用してプロジェクトを開きます。
  • ファイルから設定を読み込む - 選択したBurp設定ファイルのオプションを使用してプロジェクトを開始します。設定ファイル内のプロジェクトレベルのオプションのみ読み込まれ、ユーザレベルオプションは無視される点に注意してください。素早く選択できるよう、最近使用された設定ファイルのリストが表示されます。
