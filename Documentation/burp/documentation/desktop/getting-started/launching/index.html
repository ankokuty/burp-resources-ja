Burp Suiteの起動

Burp Suiteを起動するには、インストールされているアプリケーションのショートカットをクリックするだけです。Linuxでは、インストール時に選択したパスにショートカットがあります。

大多数のユーザは、この方法でBurp Suiteを起動します。ただし、一部のユーザはコマンドラインからBurp Suiteを起動し、実行をより細かく制御するかもしれません。

スタートアップウィザード

Burpを起動するたびにスタートアップウィザードが開きます。ウィザードは以下の手順を案内します。一部の手順はBurpの新規インストール後に初めて起動するときに実行する必要があります。

  • Burpライセンスのアクティベーション
  • プロジェクトの選択
  • 設定の選択
  • 表示設定の確認
