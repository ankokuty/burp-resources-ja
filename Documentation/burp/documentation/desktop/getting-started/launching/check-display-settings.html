表示設定の確認

Burpを初めて起動したとき、表示設定の確認にちょっと時間を割く価値があります。最初に、UIが正しく表示され、快適に読み取れるかの確認が最優先事項です。Burpでは、UIの各部分に異なるサイズのフォントが選択でき、画面解像度に応じて設定を変更もできます。

表示設定の詳細については、メインの表示オプションのドキュメントを参照してください。

まず、BurpのUIに表示されるテキストを見てください。ここには、メニュー・タブラベル・ボタンなどがあります。

メインUIのフォントサイズを変更する場合は、"オプション"タブに移動し、"表示"サブタブに移動します。"ユーザインタフェース"でフォントサイズを編集できます。その後Burpを再起動し、新しいフォントが適切か確認してください。

UIの一般的なテーマも変更できます。ダークなテーマとライトなテーマのどちらかを選択できます。

次に、"Repeater"タブに移動し、リクエストパネルにテキストを入力します。

HTTPメッセージのフォントサイズを変更する場合は、"オプション"タブに移動し、"表示"サブタブに移動します。"HTTPメッセージ表示"でフォントとフォントサイズの両方を編集できます。コンピュータで見つかった等幅フォントがすべて使用できます。

次に、Burp Repeaterに戻り、新しいフォントが適切かどうかを確認します(今回は再起動する必要はありません)。
