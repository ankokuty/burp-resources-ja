5. 次は？

おめでとうございます！ あなたはすでにBurp Suiteを使いこなし始めています。あなたは今、次について知っています:

  • Burp Proxyで、HTTP通信をインターセプトし変更する。
  • Burp Repeaterで、リクエストを再発行し脆弱性を探す。
  • Burp Scannerで、自動で脆弱性をスキャンしレポートを作成する。
  • Webセキュリティアカデミーを利用してスキルを磨く。

しかし、これはBurp Suiteが提供するすべての機能のほんの一部に過ぎません。基本的な機能を理解した上で、以下のビデオでその他の機能を確認してみてはいかがでしょうか。
