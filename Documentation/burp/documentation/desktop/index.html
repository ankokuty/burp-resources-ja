注

他のセキュリティテストツールと同様、Burp Suite には対象システムに損害を与えるおそれのある機能があります。セキュリティテストではその性質上、一部の脆弱な対象で問題を起こす可能性のある、異常な通信を伴います。Burpを使う際は十分注意し、使用前にすべてのドキュメントを読み、テスト前に対象システムのバックアップを行い、システム管理者の許可がないシステムや障害のリスクを許容できないシステムには使用しないでください。
