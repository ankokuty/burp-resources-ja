Burp拡張

Burp拡張は、Burp Suiteの動作をカスタマイズできます。Burp拡張は、コミュニティによって開発されたものも利用できますし、自分で開発もできます。

Burp拡張を使用すると、Burp Suiteのさまざまな動作を変更できます。

  • HTTPリクエストとレスポンスの変更。
  • HTTPリクエストの追加送信。
  • Burp Suiteのインタフェースをカスタマイズし、新機能やタブを追加。
  • Burp Scannerのチェック項目を追加。
  • Burp Suiteからの情報にアクセス。

Burp拡張は、BAppストアからダウンロードできます。

拡張は、Burpのサードパーティユーザによって開発、維持されています。コミュニティが開発した拡張をBAppストアで公開する前に、セキュリティや品質についてレビューを行っています。ただし、PortSwigger Web Securityは、その品質や特定の目的に対する有用性について、いかなる保証もしません。
