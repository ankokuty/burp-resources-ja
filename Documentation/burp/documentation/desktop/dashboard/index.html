ダッシュボード

Burpのダッシュボードでは、Burpの自動アクティビティを制御や監視できます:

  • 新規スキャンをクリックすると、Webサイトのスキャンを開始できます。
  • 実行中のタスクの進捗状況を監視でき、タスク詳細ウィンドウを開くと個々のタスクのより多くの情報を確認できます。
  • 検索バーを使用してタスクを検索できます。名前変更機能と組み合わせて使用すると特に便利です。
  • 個々のタスクまたはすべてのタスクを一緒に、一時停止や再開できます。
  • タスクリスト内でタスクをドラッグして並べ替えられます。
  • タスク名をダブルクリックすると、タスク名を変更できます。
  • タスクパネル上部の歯車のアイコンをクリックすると、タスクの管理方法と実行方法を設定できます。
  • 実行中の診断タスクによる問題アクティビティを表示し、報告された脆弱性をリアルタイムで確認できます。
  • 新規ライブタスクボタンをクリックすると、ライブスキャンを設定できます。
  • イベントログを表示し、警告やその他の情報を監視できます。この情報は、ネットワーク接続や他の問題のトラブルシューティングに便利でしょう。
  • 実行中、一時停止中、終了したタスクのみを表示するよう、タスクのリストをフィルタリングできます。ライブタスク、スキャン、Intruder攻撃のみを表示するようなフィルタリングもできます。

それぞれのタスクの隅にある"ポップアウト"アイコンをクリックすると、別ウィンドウとして表示されるので、Burp Suiteの他の部分で作業している間も表示しておけます。

注

1つのサイトを複数回スキャンすると、タスクサマリーパネルには新しい問題のみが表示されます。2回目のスキャンでは新しい問題が見つからないことがあり、その場合はサマリーに問題が表示されません。
