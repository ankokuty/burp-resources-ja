ダッシュボード

Burpのダッシュボードでは、Burpの自動アクティビティの制御および監視ができます:

  • "新規スキャン"をクリックすると、ウェブサイトのスキャンを開始できます。
  • 実行中のタスクの進捗状況を監視でき、タスク詳細ウィンドウを開くと個々のタスクのより多くの情報を確認できます。
  • 個々のタスクまたは全てのタスクを一緒に、一時停止や再開できます。
  • タスクリスト内でタスクをドラッグして並べ替えられます。
  • タスクパネル上部の歯車のアイコンをクリックすると、タスクの管理方法と実行方法を設定できます。
  • 実行中の診断タスクによる問題アクティビティを表示し、報告される脆弱性をリアルタイムで確認できます。
  • "新規ライブタスク"ボタンをクリックすると、ライブスキャンを設定できます。
  • イベントログを表示し、警告やその他の情報を監視できます。この情報は、ネットワーク接続や他の問題のトラブルシューティングに便利でしょう。

それぞれのパネルの隅にある"ポップアウト"アイコンをクリックすると別ウィンドウとして表示されるので、Burp Suiteの他の部分で作業している間も表示しておけます。
