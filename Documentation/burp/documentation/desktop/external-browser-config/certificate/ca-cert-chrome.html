ChromeへBurpのCA証明書をインストール

自身のブラウザでHTTPSアプリケーションをテストするには、Burp SuiteのCA証明書をインストールする必要があります。

注

大半のユーザーは、この作業は必要ありません。代わりに、すでに設定されているBurpブラウザを使用してください。

BurpのCA証明書をインストールする前に:

  • Proxyリスナーがアクティブか確認します。
  • Burpと連携するようブラウザを設定します。

Chromeで使用するBurpのCA証明書をインストールする手順は、各OSで異なります。該当する、CA証明書のインストール手順を参照してください:

  • MacOS
  • Windows
  • Linux
