Internet ExplorerへBurpのCA証明書をインストール

BurpのCA証明書をインストールする前に、Proxyリスナーが動作しており、Burpが使われるようブラウザの設定ができているか確認します。

注

以前、Burpが生成した別のCA証明書をインストールしている場合は、新しいものをインストールする前にそれを削除する必要があります。

Internet ExplorerにBurpのCA証明書をインストールするには、次の手順に従います:

Internet Explorerアイコンを右クリックし、管理者として実行を選択します。

Burpを実行している状態で、Internet Explorerでhttp://burpsuiteにアクセスします。CA証明書をダウンロードするためのリンクがある比較的わかりやすいWebページが表示されるはずです。表示されない場合は、プロキシのトラブルシューティングページを参照してください。何が悪かったかによっては、自動的にそこに遷移するかもしれません。

CA Certificateをクリックして、独自のBurp CA証明書をダウンロードします。保存する場所をメモしておいてください。

ダウンロードしたファイルを開きます。証明書ダイアログが開きます。証明書のインストールをクリックします。証明書のインポート ウィザードダイアログで、次へをクリックします。証明書をすべて次のストアに配置するを選択し、参照をクリックします。

証明書ストアの選択ダイアログで、信頼されたルート証明機関を選択し、OKをクリックします。

次へ、完了の順にクリックします。すべてのダイアログを閉じて、Internet Explorerを再起動します。Burpを実行したまま、HTTPSのURLにアクセスしてみてください。すべて正常に動作していれば、セキュリティ警告なしでページを参照できるはずです。

Internet ExplorerからBurpのCA証明書を削除

BurpのCA証明書をInternet Explorerから削除するには:

  1. Internet Explorerアイコンを右クリックし、管理者として実行を選択します。
  2. インターネットオプションを開き、コンテンツタブで証明書をクリックします。
  3. 信頼されたルート証明機関タブで、PortSwigger CAを選択し、削除をクリックします。
  4. プロンプトが表示されるたびにはいを選択し、Internet Explorerを再起動します。
