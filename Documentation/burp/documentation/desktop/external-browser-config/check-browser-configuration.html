ブラウザのプロキシ設定の確認

  1. Burpで、Proxy > インターセプトタブに移動します。インターセプト無効ボタンをクリックすると、インターセプト有効に変わります。
  2. Proxyリスナーがアクティブか確認し、選択したブラウザが設定済みか確認します。
  3. Burpを実行している状態で、設定したブラウザを開き、ブラウザでどこかHTTPのURL(まだHTTPSは使わないでください)にアクセスします。ブラウザがリクエストの完了を待っている状態、つまりページを読み込もうとしてスタックしているように見えるはずです。これは、ブラウザが送信しようとしているHTTPリクエストをBurpがインターセプトしたためです。
  4. Burpで、Proxy > インターセプトタブに移動します。インターセプトタブでは、インターセプトされたHTTPリクエストがメインパネルに表示されます。
  5. Proxy > インターセプトタブで、ボタンがインターセプト有効になっているでしょう。クリックするとインターセプト無効に変わり、Burpからリクエストが放出されます。
  6. ブラウザに戻ります。これで、通常のブラウジング時と同じように、要求されたページが読み込まれ表示されます。

上記のすべてができた場合、Burp Suiteで外部ブラウザを使用するための必須設定が完了しています。ただし現時点では、HTTPだけを使用するWebアプリケーションのみをテストできます。外部ブラウザを使用してHTTPS URLにアクセスしようとすると、接続がブロックされるでしょう。したがって、HTTPSを使用するアプリケーションをテストできるように、BurpのCA証明書をインストールする最後の追加手順を強く推奨します。
