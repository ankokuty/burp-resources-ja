Burpと連携するようChromeを設定する - Windows

ChromeでBurpを使用する場合は、プロキシ設定を行う必要があります:

  1. Chromeを開き、カスタマイズ (ハンバーガー) メニューに移動します。
  2. 設定を選択し、システムメニューを開きます。

  3. パソコンのプロキシ設定を開くをクリックします。プロキシ設定ウィンドウで、プロキシサーバの設定を行えます。

  4. 設定を自動的に検出するやセットアップスクリプトを使うがオフになっていることを確認します。
  5. プロキシサーバーを使うをオンに設定します。
  6. アドレス欄に、Burp Proxyのリスナーアドレス(デフォルトは127.0.0.1)を入力します。
  7. ポート欄に、Burp Proxyリスナーのポート番号(デフォルトは8080)を入力します。
  8. ローカル(イントラネット)のアドレスにはプロキシサーバーを使わないのチェックが外れていることを確認します。

  9. 保存をクリックします。
