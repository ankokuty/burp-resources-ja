Burpと連携するようChromeを設定する - MacOS

ChromeでBurpを使用する場合は、プロキシ設定を行う必要があります:

  1. Chromeを開き、カスタマイズ (ハンバーガー) メニューに移動します。
  2. 設定を選択し、システムメニューを開きます。

  3. パソコンのプロキシ設定を開くをクリックします。プロキシ設定ウィンドウで、プロキシサーバの設定を行えます。

  4. Webプロキシ(HTTP)プロトコルを設定します:

    • Webプロキシ(HTTP)をチェックします。
    • Webプロキシサーバ欄に、Burp Proxyリスナーのアドレスを入力します(この例では127.0.0.1)。
    • 隣の欄にBurp Proxyリスナーのポート番号を入力します(この例では8080)。
  5. セキュリティで保護された Web プロキシ (HTTPS)チェックボックスでもこの手順を行います。

  6. これらのホスト・ドメインのプロキシ設定をバイパスボックスが空になっているか確認します。

  7. OKと適用をクリックし、開いているダイアログボックスを閉じます。
