Burpと連携するようInternet Explorerを設定する

Burpでのテストに使用できるようにInternet Explorerを設定するには、次の設定手順を実行してください。

Internet Explorerで、ツールメニューに移動し、インターネットオプションをクリックします。

接続タブに移動し、LANの設定ボタンをクリックします。

設定を自動的に検出するチェックボックスがオフになっていることを確認します。

自動構成スクリプトを使用するのチェックを外します。LANにプロキシ サーバを使用するチェックボックスをオンにします。

アドレスフィールドに、Burp Proxyリスナーのアドレス(デフォルトは127.0.0.1)を入力します。ポートフィールドに、Burp Proxyリスナーのポート番号(デフォルトは8080)を入力します。

ローカルアドレスにはプロキシサーバを使用しないのチェックを外します。

詳細設定ボタンをクリックします。

すべてのプロトコルに同じプロキシサーバを使用するをチェックします。

例外フィールドをすべて削除します。

OKをクリックしてすべての設定ダイアログを閉じます。
