表示設定

表示設定では、Burp Suiteの外観を設定できます。

外観

外観設定では、次が可能です:

  • ライトテーマかダークテーマを選択する。
  • Burpメニューのフォントサイズを選択する。

注

これらの設定は、HTTPメッセージの表示に使用するフォントには影響しません。HTTPメッセージのフォントは、HTTPメッセージ表示設定で指定できます。

表示設定は、ユーザ設定です。あなたのマシンにインストールされたすべてのBurpに適用されます。

学習タブ表示

学習タブをメインバーに表示するかどうか制御します。このタブには、Burp Suiteを使用開始し新機能を発見するのに便利なチュートリアルやその他のリソースへのリンクがあります。

学習タブ表示設定は、ユーザ設定です。あなたのマシンにインストールされたすべてのBurpに適用されます。
