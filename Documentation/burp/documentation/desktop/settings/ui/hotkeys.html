ホットキー設定

ホットキーの設定では、Burp内の一般的なアクションのためのホットキーを設定できます。利用可能なアクションは、3つのカテゴリに分類されます:

  • Repeaterへ送信など、個々のHTTPリクエストやレスポンスに固有のアクション。
  • Proxyに切り替えなど、グローバルアクション。
  • 切り取り、元に戻すなど、エディタ内のアクション。

一部のホットキーがデフォルトで設定されています。

Burpの現在のホットキーを編集するには:

  • ホットキー編集をクリックし、ホットキー設定ダイアログを表示します。
  • 設定したいアクションを選択します。
  • そのアクションに割り当てたいホットキーを入力します。すべてのホットキーの組み合わせは、コントロールキー(OSXではコマンドキー)を使用する必要があります。シフトやその他の修飾キーも使用できます。
  • OKをクリックします。

注

一部のWindowsのインストールでは、Ctrl+Altキーの組み合わせがAltGrと同等に扱われる場合があります。このため、テキストフィールドでホットキーを使用しようとすると、入力した文字が表示されることがあります。

ホットキー設定は、ユーザ設定です。あなたのマシンにインストールされたすべてのBurpに適用されます。
