設定

Burp Suiteのほとんどの設定は、設定ダイアログからアクセスできます。このダイアログにアクセスするには、上部メニューの設定をクリックします。

ユーザ設定とプロジェクト設定

Burpには、2種類の設定があります:

  • プロジェクト設定は、現在のプロジェクトにのみ適用されます。設定はプロジェクトファイル自身に格納されます。
  • ユーザ設定は、あなたのマシンにインストールされたすべてのBurpに適用されます。これらは、すべてのディスクベースのプロジェクトと、一時プロジェクトに影響します。

一部の設定は、プロジェクト設定とユーザ設定の両方で定義できます。これらの設定には、このプロジェクトにのみ適用というトグルがあります。このトグルを選択すると、Burpは指定された設定をプロジェクトレベルで適用します。それ以外の場合、指定した設定はユーザ設定として扱われ、グローバルに適用されます。

このトグルにより、ユーザレベルでデフォルト値を設定し、プロジェクト単位でこれらの設定を上書きできます。

例えば、インターネットに接続するために企業内LANのプロキシをユーザ設定に設定することがあります。ただし、プロジェクトによっては、別の上位プロキシを使用する必要がある場合があります。その別の設定の詳細を、プロジェクト設定で設定できます。

設定の検索

ダイアログの横にあるナビゲーションツリーで、探している設定を見つけられます。

コンテンツをフィルタするには:

  • パネル上部の検索バーを使用します。
  • あるいは、すべて、ユーザ、プロジェクトフィルタボタンを使用します

注

Burpの最も重要な設定については、主要な設定を参照してください。
