自動バックアップ設定

自動バックアップ設定では、ディスクベースのプロジェクトについて、プロジェクトファイルの自動バックアップを設定できます。バックアップを有効にするには、自動的なバックアップ間隔[x]分後を選択し、必要な時間間隔を指定します。

注

自動バックアップを設定できるのは、ディスクベースのプロジェクトのみです。一時プロジェクトを使用している場合、自動バックアップを有効にするオプションはグレーアウトされます。

自動バックアップを実行する際のBurpの動作も制御できます:

  • スコープ内のアイテムのみ含めるかどうか。
  • バックアップ中に進捗ダイアログを表示するかどうか。
  • Burp終了時にバックアップファイルを削除するかどうか。

自動バックアップ設定は、ユーザ設定です。あなたのマシンにインストールされたすべてのBurpに適用されます。
