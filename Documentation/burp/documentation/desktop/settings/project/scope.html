スコープ設定

スコープ外リクエストの処理設定は、Burpがスコープ外リクエストを発行するかどうかを決定します。

スコープ外のすべてのリクエストを破棄するを選択すると、Burpがスコープ外のリクエストを発行しないようにします。この設定を有効にすると、ブラウザが明示的にリクエストした場合でも、スコープ外の送信リクエストはBurpによってドロップされます。

スコープ外のリクエストの定義方法について、次から選択できます:

  • 全体のスコープを使用する - この設定を選択した場合、全体のスコープ(Target > スコープで定義されている)を使用して、ドロップするリクエストを決定します。
  • カスタムスコープを使用する - この設定を選択した場合、カスタムスコープを使用して、ドロップするリクエストを決定します。Burpの標準的なURLマッチングルールを使って、スコープを定義できます。
