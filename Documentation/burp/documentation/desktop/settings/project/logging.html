ログ出力設定

ログ出力設定では、BurpのどのツールがHTTPリクエストとレスポンスをログファイルに追加できるかを設定できます。リクエスト、レスポンス、またはその両方を出力するツールを、次から選択できます:

  • Proxy。
  • Scanner。
  • Repeater。
  • Intruder。
  • Sequencer。
  • Extender。

全ツールを選択すると、リスト内のすべてのツールからログを取得できます。

ログ出力設定は、プロジェクト設定です。現在のプロジェクトにのみ適用されます。
