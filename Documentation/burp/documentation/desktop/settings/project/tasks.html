タスク設定

タスク設定は、自動化されたタスクを管理する方法を制御します。

プロジェクトファイル読み込み時のデフォルト動作

既存のプロジェクトを最初に開いたときに実行されるスタートアップウィザードで、自動タスクを一時停止するオプションのデフォルト状態を制御します。このチェックボックスをを選択すると、ウィザードで自動タスクを一時停止するがデフォルトで有効になります。

プロジェクトファイル読み込み時のデフォルト動作設定は、ユーザ設定です。あなたのマシンにインストールされたすべてのBurpに適用されます。

スケジュールタスク

タスクスケジューラを使用すると、自動化されたタスクの実行を、指定した時間に一時停止や再開ができます。

スケジュールタスク設定パネルでは、自動化されたタスクを一時停止や再開するタイミングを定義するルールを、追加や編集できます。これらのルールは、一度だけ適用、または一定期間ごとに繰り返し適用もできます。

関連ページ

スケジュールタスクの詳細は、スケジュールタスクを参照してください。

スケジュールタスク設定は、プロジェクト設定です。現在のプロジェクトにのみ適用されます。
