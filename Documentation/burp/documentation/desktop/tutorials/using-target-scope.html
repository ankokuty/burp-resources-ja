Burp Suiteのターゲットスコープの使用

ターゲットスコープを使用すると、現在のテストで関心のあるものをBurp Suiteに伝えられます。ターゲットスコープを定義すれば、今すぐテストする必要があるものだけを対象にし、作業範囲外のコンテンツや退屈または壊れやすいコンテンツを除外できます。ターゲットスコープを定義すると、パフォーマンスとメモリ使用量も改善します。Burp Suiteのツールの多くは、スコープ内のアイテムのみを対象とするように設定できます。

このチュートリアルでは、ターゲットスコープに含めるURLを設定する方法と、含めないURLを設定する方法の両方を説明します。また、一部のBurpツールを使用して、スコープ内のアイテムのみを表示/処理する方法も説明します。
