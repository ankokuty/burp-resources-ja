Burp Suiteでのライブタスクの使用

ライブタスクは、特定のBurp Suiteツールからのトラフィックを処理し、定義されたアクションを実行するために使われます。ライブタスクは、Burp Suiteツール(Proxy、Repeater、Intruderなど)からのトラフィックをスキャンし、診断し、サイトマップに追加するために、最も頻繁に使用されます。このチュートリアルでは、ライブタスクを作成し動作させる方法や、一般的なテスト機能を実行するよう事前定義されているタスクを選択する方法を紹介します。
