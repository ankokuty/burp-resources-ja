Burp Suiteプロジェクト設定の使用

Burp Suiteは柔軟に設定が可能で、プロジェクトごとにさまざまな設定ができます。このチュートリアルでは、他のプロジェクトで使用できるよう設定をファイルに保存する方法、または単一のプロジェクトに対して複数の設定を保持する方法を説明します。また、新しいプロジェクトでデフォルトで優先される設定を行う方法についても学びます。

注

このビデオの作成後、Burp Suiteの設定を再設計しました。従来のプロジェクトオプションは、ユーザレベルとプロジェクトレベルの両方の設定を一度に行える、設定ダイアログに変わりました。しかし、設定ファイルの保存と読み込みに関する情報は、現在でも有効です。

Burp Suiteで設定がどのように機能するかについての詳細は、設定のドキュメントを参照してください。
