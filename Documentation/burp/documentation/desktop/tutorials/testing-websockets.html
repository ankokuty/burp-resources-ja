Burp Suiteを使用したWebSocketのテスト

WebSocketは、双方向の非同期通信をサポートする永続的な接続です。Burp Suiteには、WebSocketのセキュリティテストをサポートするための豊富なツールセットがあります。このチュートリアルでは、WebSocketメッセージをインターセプト、表示、編集する方法を説明します。また、WebSocket接続を開始するメッセージを操作する方法についても説明します。
