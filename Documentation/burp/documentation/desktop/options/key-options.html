重要な設定オプション

Burpには、対象アプリケーションが必要とする方法で連携できるようにする、豊富な設定オプションがあります。このセクションでは、最も必要とされる可能性の高い、主要な設定オプションの概要を簡単に説明します。

HTTPメッセージの表示に使用するフォントや文字セット、Burp UI自身のフォントを設定できます。

ターゲットスコープ設定は、現在注目していて、攻撃をする意思があるアイテムをBurpに伝えます。テストの初期段階で設定すべきで、Proxy履歴、Targetサイトマップに表示されるアイテム、Proxyがインターセプトするメッセージ、スキャンされるメッセージを制御できます。

アプリケーションサーバがプラットフォームレベル(HTTP)認証を使用している場合、Burpが自動的に認証するよう設定します。

多くのアプリケーションには、セッション終了応答、リクエストトークン、状態が維持された多段階のプロセスのような、自動または手動テストを実行しづらくする機能があります。セッションハンドリングルールとマクロを組み合わせて、これらの状況にシームレスに対応できるようBurpを設定できます。

指定した時間帯だけテストをするよう、決まった時間や間隔でタスクをスケジュールするようBurpを設定できます。
