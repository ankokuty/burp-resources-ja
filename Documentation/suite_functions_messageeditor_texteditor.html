テキストエディター

テキストエディターは、HTTPメッセージエディターで、リクエストとレスポンスを生形式で表示する際や、その他Burpがプレーンテキストのコンテンツを表示する際に使用されます。  

構文解析

HTTPリクエストとレスポンスの構文は自動的に色付けされ、リクエストのパラメーターやレスポンスのHTML要素などの気になるアイテムを強調表示します。メッセージ表示オプションで、この挙動とフォントを設定できます。

構文の色づけが有効になっている場合、場所によっては構文アイテムのデコードされた値を表示するマウスオーバーポップアップも表示します。HTTPリクエストの場合、ポップアップはURLデコードを行い、レスポンスの場合はHTMLデコードを実行します。

ホットキー

テキストエディターは、様々な一般的な操作でホットキーをサポートしています。これらは、ホットキーオプションで設定でき、テキストエディターに関連するデフォルトのホットキーは次の通りです:

  • Ctrl + A, 全て選択
  • Ctrl + X, 選択したテキストをカット
  • Ctrl + C, 選択したテキストをコピー
  • Ctrl + V, ペースト
  • Ctrl + S, メッセージ全体から選択したテキスト検索して強調表示
  • Ctrl + Z, 最後の編集を元に戻す
  • Ctrl + Y, 最後に元に戻した編集をやり直す
  • Ctrl + U, 選択したテキストをURLエンコード (デコードはShiftキーも押す)
  • Ctrl + H, 選択したテキストをHTMLエンコード (デコードはShiftキーも押す)
  • Ctrl + B, 選択したテキストをBase64エンコード (デコードはShiftキーも押す)
  • Ctrl + ←, 前の単語に移動
  • Ctrl + →, 次の単語に移動
  • Ctrl + ↑, 前の段落に移動
  • Ctrl + ↓, 次の段落に移動
  • Ctrl + Home, メッセージの先頭に移動
  • Ctrl + End, メッセージの末尾に移動
  • Ctrl + Backspace, 前の単語を削除
  • Ctrl + Del, 次の単語を削除

テキスト検索

テキストエディターの下部には、表示されたテキスト内から文字列をすばやく見つけるための検索バーがあります。検索ボックスに入力すると、エディターはテキスト内の一致するアイテムを自動的に強調表示します。"<"と">"ボタンを使用して、前または次の一致箇所に移動ができます。"+"ボタンは次のオプションを表示します:

  • 大文字小文字を区別 - 検索で大文字と小文字を区別するかどうかを指定します。
  • 正規表現 - 検索語が正規表現かリテラル文字列かを指定します。
  • テキスト変更時にマッチした箇所に自動スクロール - 新しいテキストが表示されたときに、最初に強調表示された一致箇所までテキストエディターが自動でスクロールするかどうかを指定します。これは例えば、Proxy履歴でレスポンスから特定の文字列を探してアイテムを順に見ている際に便利です。このオプションを選択して新しいアイテムを選択すると、最初の検索一致箇所に自動で表示がスクロールします。

検索の強調表示に加え、Burpツールの中には、リクエストとレスポンスに独自の強調表示を適用するものがあります。例えば、Burp Scannerは問題の報告内で、HTTPメッセージの関連部分を強調表示します。検索機能を使用していない場合は、"<"と">"を使用して、ツールが生成した強調表示間で選択範囲を移動できます。
